This repository contains my work for the **Winters in Data Science (WiDS) 2025** project at **IIT Bombay**, focused on understanding and building **Reasoning Large Language Models (LLMs)** from scratch.

## üìñ About the Project

The goal of this project is to build a strong foundation in the concepts that power modern reasoning-capable AI systems. The journey progresses from fundamental Machine Learning concepts through Reinforcement Learning algorithms, which form the backbone of techniques like **RLHF (Reinforcement Learning from Human Feedback)** used in training reasoning models.

## üìÇ Repository Structure

The notebooks are organized in a sequential learning path:

| # | Notebook | Description |
|---|----------|-------------|
| 1 | [`ml basics.ipynb`](ml%20basics.ipynb) | Introduction to core Machine Learning concepts and algorithms |
| 2 | [`sklearn basics.ipynb`](sklearn%20basics.ipynb) | Hands-on with Scikit-learn library for ML implementations |
| 3 | [`digit classifier.ipynb`](digit%20classifier.ipynb) | Building a digit classification model (MNIST dataset) |
| 4 | [`multi-arm bandits.ipynb`](multi-arm%20bandits.ipynb) | Exploration vs. Exploitation using Multi-Armed Bandit algorithms |
| 5 | [`recycling robot.ipynb`](recycling%20robot.ipynb) | MDP formulation with the Recycling Robot environment |
| 6 | [`car rental problem.ipynb`](car%20rental%20problem.ipynb) | Dynamic Programming with Jack's Car Rental problem |
| 7 | [`random walk problem.ipynb`](random%20walk%20problem.ipynb) | Temporal Difference (TD) Learning methods |
| 8 | [`cliff waling problem.ipynb`](cliff%20waling%20problem.ipynb) | Comparing SARSA and Q-Learning algorithms |
